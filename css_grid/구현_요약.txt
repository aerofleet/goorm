1. style.css에 @media 쿼리 작성

- 데스크탑(>=1024), 태블릿(769~1023), 모바일(<=768) 기준으로 스타일 분기

2. 모바일 UI 최적화

- 헤더/메뉴/role 영역을 세로 정렬로 변경

- 폰트 크기 및 패딩 조정

- 그리드 영역(.highlights, .skill-grid)을 1열로 변경

3. 태블릿 UI 조정

- 폰트/여백을 중간 사이즈에 맞춰 조정

- auto-fit/minmax로 카드/그리드가 자연스럽게 줄바꿈 되도록 처리

4. 레이아웃 검증

- 브레이크포인트(1024/768)에서 레이아웃 전환 확인

- 오버플로우(가로 스크롤), 텍스트 잘림, 버튼/링크 클릭 영역 확인
